Detailed Description

Provides an easy way to list clickable buttons. Currently, the class uses "Font Awesome 5 Free" for its icons.

To use:

NavBar* navbar = new NavBar(widget, 50, 100);
navbar->addItems("\uffff", "Button");

This will instantiate the bar under the widget with a minimum width of 50 and maximum width of 100. Then it will add a button with the icon "\uffff" from the "Font Awesome 5 Free" font and a label "Button".
